#### Info:
 
 PandeMaths is a free software tool that simulates a mathematical model of pandemics.

#### Installing:

 This tool runs on many platforms and it requires Python (3.x.y).

#### Executing:
  
  python pandemaths.py (or python3 pandemaths.py)
